## 🛠️ Technical Deep-Dive (Cheat Sheet)

If someone asks "How does it work?", use these points:

### 🚀 High-End Tech Stack
- **Next.js 16 & React 19:** Utilizing the absolute latest in React Server Components for near-instant load speeds.
- **Tailwind CSS 4:** Using the brand new configuration-less engine for hyper-efficient styling.
- **Framer Motion:** All "reveal-on-scroll" animations are handled via a custom `SectionWrapper` component.

### 🎨 Design Details
- **Bento Grid Layout:** Inspired by Apple/Framer, the grid is fully responsive and reflows for mobile.
- **Glassmorphism:** Using `backdrop-filter: blur(8px)` and a subtle SVG noise filter on the background for a "paper-like" premium texture.
- **Micro-interactions:** Custom CSS transitions for the spotlight hover effect on the bento cards.
